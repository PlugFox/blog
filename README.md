## Features and Roadmap

- ✅ Frontend with Dart
- ✅ Backend with Dart
- ❌ Medium RSS parsing by authors
- ❌ SQLite database
- ❌ Pagination
- ❌ Search by tags, authors, titles.
- ❌ POST callbacks
- ❌ Dockerfiles
- ❌ Docker Compose & Stack
- ❌ CI/CD deployment with GitHub Actions
